# API

## Initialiser la base de données

```shell
php mvc db:migrate
```

## Créer un nouveau modèle

```shell
php mvc model:create NomDuModele
```

## Créer un nouveau controller

```shell
php mvc controller:create NomDuControler
```

## Lancer le projet

```shell
php mvc serve
```
